# FakeDigger（フェイクディガー）— デジタル版

ボードゲーム「FakeDigger（宝石発掘バトル）」をブラウザで遊べるデジタル版として実装したものです。
あなた（プレイヤー1）が **AI相手** に対戦します。インストール不要、`index.html` を開くだけで動きます。

## 遊び方

1. `index.html` をブラウザ（PC / スマホ）で開く
2. プレイ人数（2〜4人）を選んで「スタート」
3. 自分の手番で戦略カードを選び、対象（山札・プレイヤー・手札）をクリック
4. 全員がワーカーを使い切るとラウンド終了。いずれかの山札が0枚、またはいずれかの手札が6枚になると即時終了・得点計算

## ルール概要（このデジタル版）

- 宝石は6色：**黒・白・赤・青・黄・緑**（全40枚、黒は12枚）
- 得点：自分の**ターゲット色 = +3点**、**白 = +1点**、**黒 = −2点**、その他 = 0点
- 宝石は公式のカードアートワーク（6色の結晶イラスト＋FAKE DIGGER 裏面）を使用。手札・判明した山札・得点画面で実際の宝石カードが表示される（画像はHTMLにインライン埋め込み＝単一ファイルで完結）
- 各山は全カードを少しずつ重ねて表示（HSM 方式）。うら面には **DAI ヒント**（あり得る色）が●で表示され、調査・鑑定で見た色は結晶アイコンで確定表示
- **スクロールなしで1画面に収まるUI**：PCは横並び、スマホ縦画面は1カラムに再配置。どの端末でも収まらない分は全体を自動縮小して必ず1画面に収める
- 山の最後の1枚を発掘すると、そのカードの得点は **2倍**
- ボーナス：黒を除く**5色**が手札に揃うと **+10点** / 同じ色**5枚**で **+20点**

### 実装済みの戦略カード（ビギナーセット）

| 戦略 | コスト | 効果 |
|---|---|---|
| 発掘 | 1 | 山札トップを手札へ。何度でも可。最後の1枚は得点2倍 |
| 鑑定 | 1 | 他PLの宝石1枚を見る（相手が選ぶ）。使うと次R開始PLになる |
| 整地 | 1 | 山札2つを混ぜ、1〜5枚の2山に作り直す |
| 調査 | 1 | 任意の山札1つの全カードを見る |
| 埋葬 | 1 | 手札1枚を山札トップへ置く（山は5枚以下） |
| 強奪 | 2 | 他PLの宝石1枚を奪う（相手が選ぶ） |

## 物理版からの主な変更・簡略化

デジタル化にあたり、以下を調整しています（元ルールの意図は保持）。

- **DAI（ディグ・アシスト・アイコン）**：物理版はうら面アイコンから色を推理する仕組み。デジタル版では各カードに「あり得る色の集合」を持たせ、うら面に色ドットで表示することで推理要素を再現しています。
- **整地**：2つの山を統合後シャッフルし、選んだ枚数（各1〜5枚）で2山に分割します（1枚の山を作って次の発掘で2倍を狙う、といったコンボは可能）。物理版の細かな並べ替え操作は簡略化しています。
- **鑑定 / 強奪**：「対象カードは相手が選ぶ」ルールに従い、AIが対象になった場合は**自分にとって最も価値の低いカード**（黒など）を渡します。
- **難易度セット**：まずビギナーセット（上記6種）を実装。独占・捏造・保護・取引（スタンダード／エキスパート）は未実装で、拡張しやすい構造にしています。
- **終了保証**：AI同士がにらみ合って進行が止まらないよう、60ラウンドの安全上限を設けています（通常プレイでは到達しません）。

## 技術メモ

- 依存ライブラリなしのバニラ JS / HTML / CSS 単一ファイル
- ゲーム状態はカードID方式（`G.cards` が実体、山札・手札はID配列）で管理
- 隠匿情報は `card.knownTo`（そのカードのおもてを見たプレイヤー集合）で表現
